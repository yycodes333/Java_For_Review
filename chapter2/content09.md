

1. ***==不是类，没有构造方法。==***

2. 变量默认是final常量 `public static final 数据类型 常量名 = 常量` 。

3. 由于是静态所以调用这个常量可以***==用接口名称.常量==***

4. 方法默认是abstract 抽象方法（JDK8之前）。

   ***==public+ abstract+ 返回值类型 +方法名（参数列表）==***

5. 关系继承：

   ​	类 `implements` 接口

   ​	接口 `extends` 接口

   ​	类`extends` 类

6. `interface` 命名 接口

7. ***==接口不能用于实例化对象==***。

8. ***==接口不是被类继承了，而是要被类实现==***。

9. 抽象类VS接口

|     特性     | 抽象类 (Abstract Class) |           接口 (Interface)           |
| :----------: | :---------------------: | :----------------------------------: |
|   **定义**   |    `abstract class`     |             `interface`              |
| **成员变量** |      各种类型均可       | 只能是 `public static final` (常量)  |
| **构造方法** |     有 (给子类调用)     |                  无                  |
|   **方法**   |     抽象或普通方法      | 抽象方法 (JDK8+ 支持 default/static) |
| **继承关系** |   单继承 (`extends`)    |         多重继承(implement)          |
| **核心思想** |    "是" 什么 (Is-a)     |    "能" 做什么 (Has-a capability)    |
