## 课程介绍

本课程专为具备C语言基础或Java入门经验的人想要将所学知识投入实践应用，对于一个图形化界面代码内容有框架了解以及学习，以解心中所惑。

我们将摒弃晦涩的理论填鸭，采用中速稳进的教学节奏，将自己总结的结论配套参考菜鸟教程代码，精讲编程中最核心的知识难点。课程拒绝纸上谈兵，全程围绕一个涵盖所学知识点的**图形化界面（GUI）设计项目**展开实战演练，手把手带你打通从逻辑代码到交互设计的全流程。

这不仅是一门项目实战课，更是你通往高阶开发的基石。通过本课程，你将建立起规范的程序架构思维，为后续衔接**Web全栈开发**或深入**QT设计**打下坚实的框架基。



### 1.课程内容

|       **章节**       | **学习内容概况**                                             | **学习收获**                                                 |
| :------------------: | ------------------------------------------------------------ | ------------------------------------------------------------ |
|     **1. 数组**      | 数组的定义与内存结构、常见排序与查找算法、二维数组的应用（如地图/棋盘逻辑）。 | 掌握批量数据的存储与管理，锻炼基础算法逻辑，为后续处理复杂数据结构打下基础。 |
|      **2. 类**       | 面向对象核心思想（封装、继承、多态）、构造器、`this/static` 关键字、抽象类与接口。 | 建立“万物皆对象”的编程思维，学会将现实事物（如用户、商品）抽象为代码模型，构建模块化程序。 |
|   **3. 异常处理**    | `try-catch` 机制、自定义异常、`throw/throws` 关键字、异常体系结构。 | 提升代码的健壮性，学会如何优雅地处理程序运行中的错误，防止软件意外崩溃。 |
|    **4. 常用API**    | String/StringBuilder 字符串操作、Math 数学运算、System 类等。 | 熟悉标准库工具，避免重复造轮子，提升开发效率，能够快速解决项目中常见的逻辑问题。 |
|     **5. 集合**      | `List`、`Set`、`Map` 接口及其实现类（ArrayList, HashMap等）、迭代器、数据结构特性对比。 | 突破数组长度固定的限制，掌握动态管理数据（如增删改查）的能力，为管理项目中的海量对象提供容器。 |
|     **6. 泛型**      | 泛型类与泛型方法、通配符的使用、泛型在集合中的应用、类型擦除概念。 | 编写更安全、复用性更高的代码，理解类型约束机制，为后续理解复杂框架源码扫清障碍。 |
|    **7. 文件IO**     | File 类操作、字节流与字符流、缓冲流的读写                    | 掌握数据持久化技术，能够将内存中的数据（如游戏存档、软件配置）保存到本地磁盘，实现软件的“记忆”功能。 |
| **8. GUI图形化界面** | 窗体与组件（按钮、面板等）、布局管理器、**事件监听机制（核心）**、图形绘制类的应用。 | **课程高潮点**。将前面所有知识点串联，实现可视化的交互软件。理解“事件驱动”编程模式。 |
|    **9. 多线程**     | 线程的创建与生命周期、线程同步与安全锁、并发编程基础。       | 解决复杂任务处理（如界面不卡顿的同时后台下载数据），理解并发执行原理，提升程序性能与用户体验。 |



### 2.课程特点



对于做一个项目，一个人很难做到每块东西都掌握炉火纯青，所以有的知识点更多是现学现用，但最基础知识点是必要掌握的，否则效率过低，本课程所讲述知识点都是从Java中截取到极为重要的内容，通过快速上手熟知内容，对于Java学习基本框架有所理解，最后将所有知识点融入实战开发。
